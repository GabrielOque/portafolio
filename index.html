Landing Page Excursiones

Diseñé una página de aterrizaje para viajes y deportes extremos, resaltando actividades como deportes extremos, caminatas y aventuras. Usé JavaScript, HTML y Tailwind CSS para crear un diseño atractivo y dinámico, que muestra las emocionantes oportunidades de esta experiencia única.

Sistema de facturación

Desarrollé y desplegué una aplicación de escritorio utilizando tecnologías .NET y C#. La aplicación está respaldada por una base de datos SQL y se diseñó específicamente para optimizar la gestión de facturación interna de un hotel. Además, proporciona a los clientes la capacidad de reservar directamente a través del sistema. Esta solución integral mejora la eficiencia operativa del hotel al tiempo que ofrece a los clientes una forma conveniente de acceder a los servicios del hotel y realizar reservas en línea.

SwiftAdmin

Desarrollé y desplegué una plataforma administrativa para la Universidad de Envigado, que permite a docentes y administrativos gestionar archivos eficientemente. La aplicación, respaldada por un backend con Node y Express, una base de datos en MongoDB, y un frontend con React y CSS, está desplegada en Render. Esta solución integral, que cuenta con autenticación mediante JWT y validación de permisos por roles, mejora la eficiencia operativa y proporciona una interfaz intuitiva para la gestión de archivos, facilitando el acceso a servicios administrativos.
